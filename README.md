# ChatBot24 Studio — Landing Page v3.0

Полностью рабочий конверсионный лендинг для сервиса разработки чат-ботов под ключ. Версия 3.0 с CRO-оптимизацией.

## 🚀 Быстрый старт

```bash
# Клонирование
npm install
npm run dev
```

## 📁 Структура проекта

```
chatbot24-v3/
├── index.html              # 16 блоков лендинга
├── privacy-policy.html     # Политика 152-ФЗ
├── .env.example            # Шаблон переменных окружения
├── .gitignore              # Исключения Git
├── vercel.json             # Конфигурация Vercel + CSP
├── package.json            # Зависимости
├── api/
│   └── send-form.js        # API для Telegram с UTM
├── css/
│   └── style.css           # Тёмная тема, адаптив
├── js/
│   └── main.js             # UTM, формы, аналитика
└── assets/
    └── icons/              # SVG иконки
```

## ⚙️ Настройка окружения

```bash
cp .env.example .env
```

Заполните переменные:
```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
TELEGRAM_CHAT_ID=your_telegram_chat_id_here
YANDEX_METRICA_ID=your_yandex_metrika_id_here
```

## 🚀 Деплой на Vercel

```bash
npm i -g vercel
vercel login
vercel --prod
```

Добавьте переменные в Settings → Environment Variables:
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`

## 📊 Цели аналитики

| Событие | Описание |
|---------|----------|
| `form_submit` | Отправка формы заявки (после успеха) |
| `cta_click` | Клик на CTA-кнопку |
| `telegram_widget_click` | Клик на виджет Telegram |
| `scroll_50` | Прокрутка до 50% страницы |
| `case_view` | Просмотр блока кейсов (IntersectionObserver) |
| `faq_open` | Открытие вопроса FAQ |
